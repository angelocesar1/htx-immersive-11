BLOCK is the Default Value for positioning
BLOCK adds an element and then enters a return key

INLINE displays item without width and height.

INLINE-BLOCK allows you adjust the width and height

MARGIN allows you to create space in between the elements Clockwise.

PADDING allows you to adjust the inner distance (distance between content and the edge) clockwise

ABSOLUTE allows you to move this element wherever you want on the screen as long as you specify with TOP, RIGHT, BOTTOM, LEFT.
